Using This page

This website is quite simple to use. Firstly, the user will navigate to the home page where you will asked to upload your dataset from local storage. After that the user can move on towards the factors page. The factors page will have a table holding the data that was imported. The same table allows the user to edit any of their factors that they choose. After the user is satisfied will the data they may move onto the next page, the participant page. The participant page performs similarly to the factors page. The difference is the addition of a button that sends the user to the rating page for each participant. The user then fills in the necessary information regarding the participants' rating. Once that is finalized, the user can move onto the final results page. The page will present the user's flowchart the algorithm produced. There is an option to download the flowchart allowing the user to do what they please with the chart.

Participant Page

The participant page is where the entire process begins. Users will be asked to enter the participants that they would like to be considered. A participant is essentially someone who plays a role in the team.

Factor Page

The factor page is the second step in utilizing the site. The page is where the user will be asked to enter all the factors (variables) that they would like to be considered in their flowchart. They will also be asked which factors are and aren't related to each other.

Ratings Page

The rating page is the third step in the website. This page is where the user can input a participant's rating. A participants rating is the importance that the participant presents in the team.

Results page

The results page presents the flowchart that the algorithm produced using all the given information
