Help

{# Heading for the help page #}

1. To begin, the user should either upload factors and participants from the home page or click the Participants button at the top.

2. In the participants page, the user can add, edit, and delete participants to their liking. Follow the instructions to progress to the Factors Page.

3. Here, a user can choose the factors they want to use for the exercise, along with how many votes (frequency) each factor received from the participants.

4. After proceeding to the ratings page, the user is repeatedly asked if one factor affects the other.

5. Once the user has answered "yes" or "no" for each factor combination, the final results image is shown.

6. The user can export the factors and participants used, as well as the ratings, from the Home page.
